# Playlister Web

A Userscript Version of Playlister

## Prerequisites

- [ASDF](https://asdf-vm.com)

## Build

```bash
> asdf install
> npm install
> npm run package:userscript
```

## Usage

After building, copy `dist/userscript/playlister.user.js` into a userscript [manager](https://www.tampermonkey.net/).
